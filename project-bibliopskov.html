О проекте

Краткая информация о работе

Заказчик:

Муниципальное автономное учреждение культуры "Централизованная библиотечная система" города Пскова

Цель проекта:

Создание современного, удобного и функционального сайта для Центральной библиотечной системы Пскова, который бы стал цифровым представлением учреждения, помогал пользователям находить нужную информацию, привлекал внимание к библиотечным мероприятиям и способствовал популяризации чтения.

Адрес сайта

bibliopskov.ru

Год

2024

Что сделано

Основные улучшения

  1. Современный дизайн

    Разработан новый логотип организации, подобрана свежая цветовая схема и современные шрифты.

  2. Новая структура сайта

    Структура сайта была полностью переработана с целью обеспечения интуитивно понятной навигации.

  3. Адаптивность

    Сайт оптимизирован для различных устройств, обеспечивая комфортный просмотр на компьютерах, смартфонах и планшетах.

  4. Удобная система управления контентом

    В отличие от старой версии со статическими HTML-страницами, новая система позволяет легко добавлять и редактировать материалы без работы с кодом.

  5. SEO-оптимизация

    Внедрены современные SEO-плагины и оптимизирована структура контента для улучшения видимости сайта в поисковых системах.

Особенности

Дополнительная
информация о проекте

  • Руководство для сотрудников

    Чтобы обеспечить эффективную работу сотрудников библиотеки с новым сайтом, было создано подробное руководство по использованию системы управления контентом (CMS). Документ включает пошаговые инструкции с примерами, разъясняет основные функции и помогает быстро освоиться с обновленным инструментом, учитывая невысокий уровень технической подготовки сотрудников.

  • Перенос данных

    Одним из вызовов проекта стало перемещение большого количества страниц со старого сайта (500+ на момент запуска обновленной версии).
